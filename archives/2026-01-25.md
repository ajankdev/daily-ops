# Daily Digest: 2026-01-25

### Release 1.143.0
**Source:** KCC Release

> **Category:** [GCP_K8S_CORE]
> **Summary:** Release 1.143.0 of the Kubernetes Config Connector (KCC) introduces new beta and alpha resources, including ArtifactRegistryRepository, LoggingLink, MemorystoreInstance, PrivateCACAPool, and ParameterManagerParameter. It also adds new fields to existing resources, such as AlloyDBInstance, and improves reconciliation for resources like TagsTagBinding. Additionally, the release includes bug fixes and new features like structured diff reporting and improved memory management.
> **Impact:** This release is expected to improve the stability and functionality of KCC, allowing for better management of Google Cloud resources within Kubernetes clusters. The new resources and fields will provide more fine-grained control over cloud resources, while the bug fixes and improvements will reduce errors and enhance overall performance.
