### Supercharge your Cloud SQL for MySQL write performance with new optimized writes
**Source:** Google Cloud (General)

> **Category:** [GCP_K8S_CORE]
> **Summary:** Cloud SQL for MySQL Enterprise Plus edition now includes optimized writes, a set of automated features that adjust MySQL configurations based on real-time workload and infrastructure metrics to reduce write latency and increase throughput. The optimized writes feature includes five different optimizations: adaptive purge, adaptive I/O limits, scalable sharded I/O, faster REDO recovery, and adaptive buffer pool warmup.
> **Impact:** This feature can significantly improve the write performance of Cloud SQL for MySQL instances, with up to 3x better write throughput and reduced latency, making it suitable for write-intensive OLTP workloads. The optimized writes are enabled by default for all Enterprise Plus edition instances, and users can upgrade their existing instances to take advantage of this feature.
