### v7.19.0
**Source:** Terraform GCP Release

> **Category:** [OPS_STACK]
> **Summary:** The Terraform Google provider has released version 7.19.0, which includes deprecations, new resources, improvements, and bug fixes. The deprecations include the removal of `google_backupdr_restore_workload.name` and `google_public_ca_external_account_key.b64url_mac_key`. New resources have been added, such as `google_network_security_mirroring_endpoint` and `google_network_security_mirroring_endpoint_group`. Improvements include added support for network interface attachments and VLANs in compute instances, as well as new fields in various resources. Bug fixes address issues with compute, container, and public CA resources.
> **Impact:** This release may impact operations as it introduces deprecations, new resources, and changes to existing resources. Users should review the release notes to ensure compatibility with their current Terraform configurations and update their code accordingly to avoid potential breaking changes.
